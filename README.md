# Student Performance Prediction

## Overview

This project uses Machine Learning to predict student academic performance based on various factors such as study time, attendance, previous grades, family support, and more.

## Features

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Performance Prediction

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
